How to make something we can sell:
First create the base article using this form, this will be the product we use to make a product. When we have the base article we also need to add the related articles, if any. A related article might be a case or a pallet that we use to order the base article.

For example, we want to sell product X on the site. We first add the article, using this form, to create Article X. Now we need to think about how we buy Article X, do we buy them in boxes or pallets?

If we buy a case and that case has a GTIN, we add a new article (Article Y). This article is will be the case that contains, let's say 10 Article X.

Now we need to add a connection from the articles we want to order and the supplier. This is done using MerchantArticles.

For every article we can order, we should create a MerchantArticle. In our example, we only order the case (Article Y) so we create a MerchantArticle with this the GTIN for Atricle Y and the suppliers materialnumber.

If we can order Article Y from more than one supplier we would add a MerchantArticle for each supplier, all using the same GTIN but different materialnumbers.

Checklist
  1. Create base article
  2. Create realted articles (Cases, pallets etc.)
  3. Create MerchantArticles for any article we order
